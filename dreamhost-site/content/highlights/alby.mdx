Alby is the connective tissue between Lightning and the web. The browser extension implements WebLN for web-based Lightning interactions and doubles as a Nostr key signer. Alby Hub provides a self-custodial Lightning node with channel management, sub-wallets, and direct app connections.

For node operators, Alby Hub is particularly interesting — it can connect to your existing node and expose it to web applications, making your routing node useful beyond just forwarding payments.
