The Breez SDK lets any developer add Lightning payments to their app without running a full node. It's nodeless and non-custodial — funds move between Lightning and the Liquid sidechain through submarine swaps, and the SDK ships bindings for Android, JavaScript/Node.js, React Native, Go, and C#, with a free API key to get started.

This matters for the Lightning ecosystem because it removes the single biggest barrier to entry for wallet and app developers: node operations. Every app built on the SDK is another on-ramp generating real Lightning volume for the network to route.
